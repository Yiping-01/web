# 《當我飛奔向你》影劇介紹網站

這是一個單頁式影劇介紹網站範本，包含：

1. 導覽列
2. 首頁大圖
3. 劇情簡介
4. 角色卡片
5. 作品資訊
6. 劇照區
7. 影片精華

## 使用方式

直接打開 index.html 就可以預覽。

## 修改圖片方式

目前圖片位置都是佔位區塊。

之後你可以把圖片放到 images 資料夾，例如：

- images/hero.jpg
- images/poster.jpg
- images/zaizai.jpg
- images/rangrang.jpg
- images/photo1.jpg

再到 index.html 把對應的 div 改成 img 標籤。
